# dev-say

## Installation
```
npm i dev-say
```

## About dev-say

Outputs objects in a lucid maner:

```
const someObject = {
  keyA: 'Hello, Object!',
  keyB: {
		keyC: {
			someArr: [1,2,3],
			keyD: {
				someValue: 'Hello, Value!'
			}
		}
	}
};
```
